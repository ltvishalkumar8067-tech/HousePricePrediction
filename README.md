# 🏠 House Price Prediction Web App

## Project Overview

This project predicts house prices using Machine Learning based on various house features.

## Features Used

- MS SubClass
- MS Zoning
- Lot Area
- Neighborhood
- Overall Qual
- Year Built
- Gr Liv Area
- Bsmt Full Bath
- Full Bath
- Bedroom AbvGr
- Garage Cars
- Garage Area

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- GitHub

## Machine Learning Model

- Linear Regression
- Data Cleaning
- Label Encoding
- Train Test Split
